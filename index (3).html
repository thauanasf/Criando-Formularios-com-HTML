O elemento "datalist"

O "datalist" elemento especifica uma lista de opções predefinidas para um elemento. Os usuários verão uma lista suspensa das opções predefinidas à medida que inserem dados. O listatributo do elemento deve se referir ao idatributo do "datalist"elemento.

O elemento "output"

O elemento representa o resultado de um cálculo (como um realizado por um script).
